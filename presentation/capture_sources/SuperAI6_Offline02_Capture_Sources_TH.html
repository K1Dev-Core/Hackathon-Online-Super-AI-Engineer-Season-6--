SLIDE 06

Rule engine และ hard negative

แคป code นี้ไปอธิบายเทคนิคจริง

is_syn_signature = (
    frame_len == 54
    and tcp_window_size == 512
    and syn == 1
    and ack == 0
)

is_hard_negative = (
    frame_len == 56
    and tcp_window_size == 253
    and mqtt_msgtype == 13
)

กฎสร้าง evidence ก่อน จากนั้นใช้ stream context ตรวจซ้ำ และใช้ PINGRESP เป็น hard negative กัน false positive
